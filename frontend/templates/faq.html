Questions frequentes

Tout ce que vous devez savoir sur Open Always

Qu'est-ce qu'Open Always ? ▼

Open Always est une plateforme gratuite qui vous permet de discuter avec plus de 10 modeles d'IA differents, incluant GPT-4, Claude, Gemini, Llama et plus. Creee par Precieux Okitakoy (Okitakoy Inc.).

C'est vraiment gratuit ? ▼

Oui ! Open Always est 100% gratuit avec des messages illimites. Vous obtenez 5 cles API a l'inscription et pouvez en gagner plus en regardant des publicites.

Comment obtenir une cle API ? ▼

Creez un compte et une cle API sera automatiquement generee. Vous pouvez la trouver dans votre Dashboard. Utilisez-la avec l'en-tete Authorization: Bearer VOTRE_CLE.

Quels modeles sont disponibles ? ▼

GPT-4, GPT-3.5 Turbo, Claude 3, Claude 3 Opus, Gemini Pro, Gemini Ultra, Llama 3.1, Llama 3.1 70B, Mistral Large, DeepSeek V3, Cohere Command et notre propre Okitakoy AI.

Comment fonctionne l'authentification ? ▼

Vous pouvez vous inscrire par email (avec verification OTP) ou via Google OAuth. Une fois connecte, votre session reste active et votre cle API est utilisable dans vos projets.

L'API est-elle compatible REST ? ▼

Oui, l'API est RESTful. Envoyez un POST a /api/chat avec votre message et le modele souhaite. Consultez la documentation pour tous les endpoints.
